### Folder Description

-   **cuanti_secund_Canasta_Basica_TIC_2023/**\
    Contains raw data downloaded from official sources (ENCV-2022, CRC,
    MinTIC/CRC, population projections, DANE shapefiles, among others).
    These datasets support the quantitative analysis of ICT expenditures
    across regions and population groups.

-   **codigo_Canasta_Basica_TIC_2023/**\
    Stata scripts used to clean, transform, merge, and prepare
    analytical datasets.

-   **analisis_Canasta_Basica_TIC_2023/**\
    Intermediate outputs generated by the scripts, including statistical
    tables, analytical outputs, and working materials prior to the final
    report.

-   **resultad_Canasta_Basica_TIC_2023/**\
    Contains final analytical products: summary tables, maps,
    visualizations, and outputs used in the final deliverable.

## ğŸ“Œ Project Objective

The main objective of the project is to characterize the **Basic ICT
Basket (CBT)** for Colombia in 2023 by analyzing:

-   Household expenditures on ICT goods and services\
-   Penetration and accessibility of fixed and mobile connectivity\
-   Comparative cost structures using official data sources\
-   Spatial distribution of key ICT-related indicators

This repository supports the development of empirical evidence for
public policy related to **digital inclusion** and reduction of access
gaps in ICT services across Colombia.
